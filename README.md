# AC05 - Arquitetura de projetos e sistemas 

Rodamos dois containers do docker para o projeto, sendo um para o **front-end** e outro para o **back-end**.

## Requisito
- Docker

## Tecnologias

- React
- Axios
- NodeJS
- Express
- SQLite
- Docker

## Como rodar?

### Front-End
```
    docker compose up
```

### Back-End
```
    docker compose up
```
